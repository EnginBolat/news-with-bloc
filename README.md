## Run Locally

Clone the project

```bash
  https://github.com/EnginBolat/news_with_bloc.git
```

Go to the project directory

```bash
  cd news_with_bloc
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
